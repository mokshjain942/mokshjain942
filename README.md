### 🎮 Neon Surge
*A minimalist neon arcade survival game, inspired by Pieter Levels' build-in-public philosophy.*

Neon Surge is an obstacle-based survival game with power-ups and a progression system, built around a clean, futuristic visual identity. It was less about the game itself and more about the practice — working through game logic, UI/UX decisions, and performance trade-offs from scratch. It taught me that "finished" is a design decision, not a default state.
